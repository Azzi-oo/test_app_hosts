# HTTP Host Benchmark Tool

Инструмент для тестирования и бенчмаркинга HTTP-хостов с поддержкой параллельных запросов и детальной статистикой.

## Требования

- Python 3.7+
- requests==2.32.5

## Установка

1. Клонируйте репозиторий или скачайте проект
2. Создайте виртуальное окружение (опционально):
   ```bash
   python3 -m venv venv
   source venv/bin/activate  # для Linux/Mac
   # или
   venv\Scripts\activate  # для Windows
   ```

3. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

### Параметры командной строки

- `-H, --hosts` - Список хостов через запятую (взаимоисключающий с `-F`)
- `-F, --file` - Путь к файлу со списком хостов (взаимоисключающий с `-H`)
- `-T, --timeout` - Таймаут запроса в секундах (по умолчанию: 10.0)
- `-C, --count` - Количество запросов на каждый хост (по умолчанию: 1)
- `-P, --parallel` - Количество потоков для параллельного тестирования хостов (по умолчанию: 1)
- `-O, --output` - Путь к файлу для сохранения результатов (опционально)
